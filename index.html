Mosque Data Management System, Qassim 2025

Python-Based GUI Application for Mosque and Imam Organization A comprehensive Python application developed with a graphical user interface (GUI) to organize and manage data related to mosques and their assigned imams. The project emphasizes clean database operations, GUI responsiveness, and practical usability for community and religious centers.
